# Stitch Prompts - Neo-Brutalist VibeChat

Use these prompts with Stitch MCP to generate additional Neo-Brutalist designs matching the VibeChat aesthetic.

## Design System Tokens

- **Primary**: `#fed023` (Yellow)
- **Secondary**: `#ff6d8d` (Pink)
- **Tertiary**: `#9fff88` (Green)
- **Background**: `#0e0e0e` (Near-black)
- **Surface**: `#191919` (Dark gray)
- **Fonts**: Epilogue (headlines, black/900 weight), Space Grotesk (body, bold/700 weight)
- **Borders**: 4px solid black
- **Shadows**: 4px or 8px offset, no blur, black color
- **Border radius**: 0px (sharp corners, except `9999px` for pills)
- **Interaction**: `translate(4px, 4px)` on active, `transition: none` for snappy feel

## Prompts

### 1. Settings Page

```
Design a Neo-Brutalist settings page for an anonymous chat app called VibeChat.
Dark background (#0e0e0e), bold Epilogue headlines, yellow (#fed023) accents.
Use 4px black borders, hard shadows (no blur), zero border-radius.
Include sections for: Account, Notifications, Privacy, Appearance.
Use Material Symbols icons. Include toggle switches with green (#9fff88) active state.
Layout: full-width cards in a vertical stack with black borders between sections.
```

### 2. User Search / Discover Page

```
Design a Neo-Brutalist user discovery/search page for VibeChat.
Dark theme with surface-container (#191919) cards.
Include a large search input with 4px black border, yellow focus state.
Display user cards in a 3-column grid on desktop, 1 column on mobile.
Each card: avatar placeholder, username, vibe status tag, online indicator.
Use pink (#ff6d8d) for online indicators, gray (#757575) for offline.
Hard 4px black shadows, bold uppercase Epilogue text, no rounded corners.
```

### 3. Error / 404 Page

```
Design a Neo-Brutalist 404 error page for VibeChat.
Large glitched "404" text in Epilogue black weight on dark background.
Use yellow (#fed023) and pink (#ff6d8d) as accent colors.
Include a large neo-brutalist button to return home.
Add a scrolling ticker tape at the bottom with "SIGNAL_LOST" repeating.
Use 8px black shadows, 4px borders, zero border-radius.
```

### 4. Loading / Splash Screen

```
Design a Neo-Brutalist loading screen for VibeChat.
Large animated "VIBECHAT" logo in Epilogue italic black, yellow color.
Pulsing green (#9fff88) dot indicators.
Progress bar with 4px black border, yellow fill, hard shadow.
Dark background (#0e0e0e), no rounded corners.
Add a subtle halftone dot pattern overlay at 10% opacity.
```

### 5. Notification Panel

```
Design a Neo-Brutalist notification dropdown panel for VibeChat.
Fixed position panel, 400px wide, dark surface (#191919) background.
4px black border, 8px hard shadow.
Notification items: icon + text + timestamp, separated by 2px black borders.
Highlight unread with yellow (#fed023) left border accent.
"Mark all read" button in secondary pink (#ff6d8d) neo-brutalist style.
Bold uppercase text, Material Symbols icons, zero border-radius.
```

### 6. Voice Chat Overlay

```
Design a Neo-Brutalist voice chat overlay/modal for VibeChat.
Dark semi-transparent backdrop, centered modal with 4px black border.
Show active speakers with animated green (#9fff88) wave indicators.
Mute/unmute, deafen, disconnect buttons in neo-brutalist style.
Participant grid with avatar circles, usernames, speaking status.
Use hard shadows, bold Epilogue text, primary yellow accents.
```
